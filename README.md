# NoteNest Student

The ultimate AI-powered note organizer for students. Scan notes, auto-organize into subjects/chapters/topics, generate quizzes, and more.

## Features
- Built-in document scanner (CamScanner-style with OCR)
- AI auto-filing and quiz generation (GPT-4o)
- Real-time Firebase sync
- Beautiful glassmorphic UI
- PWA installable

## Tech Stack
- Next.js 15 + TypeScript
- Firebase (Auth, Firestore, Storage)
- Tesseract.js (OCR)
- OpenAI (AI features)
- Tailwind CSS + Framer Motion

## Live Demo
https://notenest-191cc.web.app

## Setup
1. Clone: `git clone https://github.com/umayer16/NoteNest.git`
2. Install: `npm install`
3. Env: Copy `.env.example` to `.env.local` and add keys
4. Run: `npm run dev`
